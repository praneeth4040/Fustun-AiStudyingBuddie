# Contributor Covenant Code of Conduct

We expect all contributors to follow the Contributor Covenant Code of Conduct. Please be respectful and constructive.

Summary
- Be respectful and professional.
- Do not harass or discriminate.
- Report unacceptable behavior to the repository maintainers.

For the full Code of Conduct, please refer to https://www.contributor-covenant.org/version/2/0/code_of_conduct/
